## 课程资源

- **在线视频**：提供丰富的教学视频，方便学员随时随地学习。
- **在线演示**：提供在线Demo演示，体验项目真实效果。受限于服务器硬件资源，暂时只部署了第一阶段的功能。见<http://207.t.imooc.com/>。
- **代码仓库**：建立代码仓库，共享课程代码和项目源码，方便学员参考和实践。见`samples`目录。
- **文档资料**：提供详细的课程文档、技术手册和项目文档，帮助学员深入理解课程内容。见`docs`目录。

通过以上课程设计，学员将逐步掌握 Java 全栈开发的核心技术和思维，从单体项目开发过渡到微服务项目开发，同时融入前后端分离架构、AI等新技术，提升自身的职场竞争力。
